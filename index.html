Frequently Asked Questions?

What is the difference between var, let, and const?
Var means global variable.It can be accessed from any line of the code.Its not a block scope variable.Its value can be reassigned and changed.

Let is block scope variable.It can be accessed only in its defined block.I mean inside {}.Its value can't be reassigned in its block scope but can be re-declared outside the scope.

const means constant value.It's block scoped.It's value can't be re-declared and re-assigned.
What is the difference between map(), forEach(), and filter() ?
map() selects an array,do some work and returns a new array when the work is done. foreach() does the work but do not returns an array. filter() and map() is simillar but the main difference is in use case.we use filter() filter a array.And filter() also returns an array with filtered values.

How JavaScript Promises work?
When we try to fetch from an api it gives a promise.Promise means a commitment that the api will give us the data.promise has 3 state. 

      1. Pending = It means the data is loading.      
      2. Resolve = Data is loaded and can ude data now.     
      3. Rejected = The api call was not successfull.So the data could not load.
How closures work in JavaScript?
When we create a function and inside that function there is another function that can access the variables or data from the main function.And when we close the container functiion we can still use the inner function and use the info of the container function.
